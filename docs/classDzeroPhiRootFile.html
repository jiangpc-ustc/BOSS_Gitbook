Date
October 25th, 2018
Remarks
DEVELOPMENTAL

The DzeroPhiRootTree is derived from the general BOSSRootTree class. It can be used to easily load a ROOT file generated in the jopOptions_ana (analysis) step of the BOSS framework. As opposed to the more BOSSRootTree class, the structure of this class needs to match that of the analysis you are working with.
