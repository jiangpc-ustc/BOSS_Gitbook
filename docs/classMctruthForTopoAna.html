Detailed Description

Helper class for the topoana package.

It is essential to run load this algorithm along with your own algorithm if you want to run topoana.exe over the output of your initial event selection. This algorithm (MctruthForTopoAna) will store the relevant 'MC truth' information.

Remarks
January 21st, 2019: *This procedure has been implemented in the TrackSelector and does not have to be loaded anymore.
