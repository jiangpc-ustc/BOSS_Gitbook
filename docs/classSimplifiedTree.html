Detailed Description

A container class for a TTree that allows easy access to its branches.

Addresses are set automatically upon construction.

Automatically book addresses for all branches of a TTree.

The branches are accessible by name (use Get_<type>, though you have to the type to use this).

Parameters
fTreeThe TTree that you want to load.
printWhether or not to print the TTree names and its branches+types. Give true to the SimplifiedTree constructor for debugging purposes in particular.

The data type of a branch can be determined from the last character of its title. See https://root.cern.ch/doc/master/classTTree.html#a8a2b55624f48451d7ab0fc3c70bfe8d7 for the labels of each type.

void SimplifiedTree::DrawAndSaveAllMultiplicityBranches ( const TString &  logScale = "")

Draw the distributions of all branches of the underlying TTree if its name starts with "mult".

This function additionally ensures that the bin width is set to 1 and that the histograms are drawn in "E1" mode (see https://root.cern.ch/doc/master/classTHistPainter.html).
