Remarks
DEVELOPMENTAL

The RhopiRootChain is derived from the general BOSSRootChain class. It can be used to easily load a ROOT file generated in the jopOptions_ana (analysis) step of the BOSS framework. As opposed to the more BOSSRootChain class, the structure of this class needs to match that of the analysis you are working with.
