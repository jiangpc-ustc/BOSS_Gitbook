NTuplePtr DzeroPhi::BookNTuple ( const char *  tupleName)
private

Helper function that allows you to create pair of a string with a NTuplePtr.

This function first attempts to see if there is already an NTuple in the output file. If not, it will book an NTuple of

Parameters
tupleNameThis will not only be the name of your NTuple, but also the name of the TTree in the output ROOT file when you use NTuple::write(). The name used here is also used as identifier in NTuplePtr map fNTupleMap. In other words, you can get any of the NTuplePtrs in this map by using fNTupleMap[<tuple name>]. If there is no NTuple of this name, it will return a nullptr.
