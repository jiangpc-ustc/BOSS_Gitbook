◆ fDoFit4c

bool DzeroPhi::fDoFit4c
private

NTuples are like vectors, but its members do not necessarily have to be of the same type.

In this package, the NTuples are used to store event-by-event information. Its values are then written to the output ROOT file, creating a ROOT TTree. In that sense, each NTuple here represents one TTree within that output ROOT file, and each NTuple::Item represents its leaves. The name of the leaves is determined when calling NTuple::addItem. Note that the NTuple::Items have to be added to the NTuple during the TrackSelector::initialize() step, otherwise they cannot be used as values! This is also the place where you name these variables, so make sure that the structure here is reflected there! Package property that determines whether or not to perform and record a 4-constraint Kalman kinematic fit.
