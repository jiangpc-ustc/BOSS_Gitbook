Detailed Description

This is a base class for fit specific objects that you use for fitting invariant mass spectra.

Author
Remco de Boer 雷穆克 (r.e.d.nosp@m.eboe.nosp@m.r@stu.nosp@m.dent.nosp@m.s.uu..nosp@m.nl or remco.nosp@m..de..nosp@m.boer@.nosp@m.ihep.nosp@m..ac.c.nosp@m.n)
Date
November 26th, 2018

This base class provides its derivate classes with the members and methods necessary to load the spectrum and the ReconstructedParticle object of the particle that you want to reconstruct. Basic RooFit objects such as RooDataHist that are needed when fitting whatever the function you are fitting are initialised through this class as well.
