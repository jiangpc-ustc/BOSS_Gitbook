Detailed Description

Container object that is to facilitate the booking procedure for NTuple::Tuples and their NTuple::Items.

See this page for the motivation. This function also automatically declares a corresponding JobSwitch property.

Remarks
This object can only be used within a derived class off the Gaudi Algorithm class because its fTuple requires to be booked by it (using e.g. TrackSelector::BookNTuples).

◆ AddItem()

void NTupleContainer::AddItem ( const std::string &  item_name)

Easier and expanded version of NTuple::Tuple::addItem.

  1. Abort if fTuple has not been booked.

  2. Abort if the perform job switch property has been set to false.

  3. Create an NTuple::Item using the items mapping.

  4. Create an NTuple::Item using the items mapping to the fTuple and to the map of items, if allowed by the perform job switch.
