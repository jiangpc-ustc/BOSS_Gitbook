1 # Organisation of the IHEP server
2 
3 The IHEP server runs on [Scientific Linux CERN](https://linux.web.cern.ch/linux/scientific.shtml) \(SLC\). The server offers several versions. Usually, people use either SLC5, SLC6, or SLC7. The domain names for these are `lxslc7.ihep.ac.cn`, where the `7` in this case refers to SLC7. If you are running on Linux or a Linux terminal, the server can be easily accessed using:
4 
5 ```text
6 ssh -Y <your user name>@lxslc7.ihep.ac.cn
7 ```
8 
9 Here, the option `-Y` ensures _X11 forwarding_, allowing you to open graphical applications.
10 
11 In Windows, there are some nice tools that allow you to access the server. First of all, to be able to use SSH, use will either have to use [PuTTY](https://www.putty.org/) or more extensive software like [Xmanager](https://www.netsarang.com/products/xmg_overview.html). You can also just search for some Linux terminals for Windows. In addition, I would recommend you to work with the \(S\)FTP client [WinSCP](https://winscp.net/eng/index.php). It allows you to easily navigate the file structure of the IHEP server and to quickly transfer---even synchronise---files up and down to your own computer.
12 
13 When you have logged into the server, you usually start in your home \(`~`\) folder. Move to the root of the server \(`cd /`\) and you'll see that is a large number of other directories. A few of these directories contain space that is assigned to your user account. Here is an overview:
14 
15 | Path | Data quotum | Max. number of files | Remark |
16 | :--- | :---: | :---: | :---: |
17 | `/afs/ihep.ac.cn/users/<letter>/$USER` | 500 MB | NA | home \(`~`\) |
18 | `/besfs/users/$USER` | 50 GB | 300,000 | |
19 | `/ihepbatch/bes/$USER` | 200 MB | NA | |
20 | `/workfs/bes/$USER` | 5 GB | 50,000 | no `hep_sub` available |
21 | `/scratchfs/bes/$USER` | 500 GB | NA | max. 2 weeks |
22 
23 Of course, here, and in what follows, `$USER` refers to your user name.
24

29 * [Raw data files](https://docbes3.ihep.ac.cn/~offlinesoftware/index.php/Raw_Data)
30  * `/bes3fs/offline/data/raw`
31  * `/besfs2/offline/data/raw` \(only rounds 3 and 6\)
32  * `/besfs/offline/data/randomtrg` \(random trigger data\)
33 * [Reconstructed data sets](https://docbes3.ihep.ac.cn/~offlinesoftware/index.php/Production)
34  * `/besfs3/offline/data/`
35  * `/besfs2/offline/` \(older versions\)
36 
37 These directories will be important later in this 'tutorial'.
