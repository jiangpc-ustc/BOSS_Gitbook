# BaskeAnaTool

A very useful, Python-based tool is being developed by Ma Xin-Xin. The package works independent of BOSS, but facilitates for instance MC simulation. The package can be obtained from GitHub:  
[https://github.com/xxmawhu/BaskeAnaTool](https://github.com/xxmawhu/BaskeAnaTool)

Unfortunately, instructions are in Chinese, but the commands speak for themselves ;\)
